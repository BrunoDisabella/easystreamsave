# Agent Brief - Easy Stream Save

## Mision

Trabajar de forma autonoma para llevar Easy Stream Save desde dominio comprado hasta una extension Chrome monetizable.

## Archivos Fuente

Antes de trabajar, leer:

- `BUSINESS_PLAN.md`
- `ROADMAP.md`
- `WORKLOG.md`
- `SEO_AND_GROWTH.md`
- `COMPETITOR_PRICING.md`

## Objetivo Comercial

Monetizar una extension Chrome en ingles para detectar y descargar videos/streams disponibles en la pagina actual, con UX limpia, sin anuncios invasivos y privacidad clara.

## Restricciones

- No prometer DRM bypass.
- No prometer descargar YouTube/Netflix/Disney de forma universal.
- No hacer pagos, publicar extensiones, enviar emails ni cambiar configuraciones externas sin permiso de Bruno.
- No tocar marcas Los Gurises/Bruno: marca separada.
- No exponer secretos ni tokens.

## Modo de Trabajo Autonomo

En cada ejecucion:

1. Leer `WORKLOG.md` para saber lo ultimo hecho.
2. Leer `ROADMAP.md` para elegir la siguiente tarea desbloqueada.
3. Avanzar una unidad concreta de trabajo que no requiera intervencion de Bruno.
4. Actualizar `WORKLOG.md`.
5. Actualizar `BUSINESS_PLAN.md` si hubo decision o cambio estrategico.
6. Mejorar `ROADMAP.md` cuando aparezcan nuevos objetivos medibles o procesos profesionales.
7. Si trabaja SEO/growth/pricing, actualizar `SEO_AND_GROWTH.md` o `COMPETITOR_PRICING.md`.
8. Avisar a Bruno:
   - que se trabajo;
   - que archivo/codigo se modifico;
   - que falta;
   - si necesita accion de Bruno.

## Prioridad Actual

1. No insistir con dominio/Cloudflare hasta que Bruno retome ese punto.
2. Probar el MVP con preview en Chrome en sitios no DRM.
3. Anotar casos donde la descarga funciona pero el preview no carga por CORS/headers.
4. Preparar screenshots y descripcion larga para Chrome Web Store.
5. Crear backlog de experimentos SEO y captacion.
6. Avanzar contenidos SEO/landing que no requieran cambios DNS.
