Descargar para Chrome

Instala la extension, abre una pagina con video y guarda medios disponibles con un flujo mas limpio.

Easy Stream Save esta pensada para medios accesibles desde el navegador. Detecta fuentes expuestas, muestra contexto util y mantiene clara la accion de descarga.

Recibir acceso de lanzamiento Comparar planes
1

Abre una pagina con medios

Entra a una pagina que exponga fuentes MP4, WebM, M3U8, MPD, MOV o M4V a Chrome.

2

Revisa las fuentes detectadas

Usa el popup para ver previews, tipo de archivo, pistas de tamano y ordenamiento mas claro.

3

Descarga lo disponible

Guarda medios accesibles para uso personal y legal. No se saltan videos protegidos o con DRM.

Lanzamiento en Chrome Web Store

La publicacion publica en Chrome Web Store esta en preparacion. El plan gratis quedara disponible primero; el cobro Pro se activara cuando el sistema de cuentas y suscripciones este listo.
